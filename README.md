# Theme for body-works

## Development instructions

Run `Grunt` in watch mode to generate new assets on the fly:

```
grunt watch
```

Run `Grunt` in default mode to create responsive images:

```
grunt default
```

### Translations

Use `Poedit` with domain `body_works`.
